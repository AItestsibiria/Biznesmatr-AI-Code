# CRM-модуль — Бизнесметр

Интегрирован в основную платформу через Supabase.

## Сущности

- **Лиды** (`crm_leads`) — контакты заинтересованных покупателей/арендаторов
- **Задачи** (`crm_tasks`) — напоминания, звонки, показы
- **Воронка** — new → contacted → qualified → lost/won

## Функции

- Привязка лида к объекту
- AI-скоринг лида (GPT-4o анализ активности)
- Email/SMS уведомления (Resend + SMS.ru)
- История взаимодействий
